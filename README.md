#### 介绍

#### 软件架构

基于javaweb的SSM架构，数据库用到了关系型数据库mysql和非关系型数据库mongdb


#### 安装教程

1. git clone 项目地址
2. web项目导入idea
3. idea集成Tomcat，在idea中运行项目
